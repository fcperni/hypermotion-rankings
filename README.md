# hypermotion-rankings

Proyecto para controlar los rankings personales de la Liga Hypermotion de un grupo de amigos, comparándolo con el dato real.

El formato del fichero predictions.csv tiene que ser el siguiente:

```
name,team,predicted_position
Ana,Real Madrid,1
Ana,Barcelona,2
Ana,Atletico Madrid,3
Ana,Real Sociedad,4
...
John,Real Madrid,2
John,Barcelona,1
John,Atletico Madrid,4
John,Real Sociedad,3
...
```

Futuras Mejoras:
* Ahora mismo está a fuego que utilice el ranking de La Liga Hypermotion. Esperemos que, en el futuro, haya que usar los de La Liga EA.
* Manejar la lista de mapeos de equipos de otra manera.
* Mostrar los puntos en la lista de standings.
* Aplicación multidioma.
